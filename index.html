Tentang CekKopit

CekKopit adalah sebuah situs sederhana yang menampilkan statistik data kasus COVID-19 di Indonesia untuk hari ini dan 7 hari kebelakang. CekKopit mengambil data dari mirror API pemerintah dan diatur sedemikian rupa agar mudah dibaca.

Nama CekKopit sendiri diambil dari nama virus COVID-19 dan diubah sedikit menjadi Kopit yang biasa masyarakat sebut, hehe :D.

Kenapa Buat Lagi? 'Kan ada situs portal dari pemerintah?

Ada tiga alasan saya membuat situs ini:

  1. Karena situs tersebut lambat untuk diakses di daerah saya (atau setidaknya rumah saya),
  2. Untuk menyediakan kembali situs portal yang hanya berisi data dan statistik, sederhana, tanpa berita dan kontak segala macem, dan
  3. Hobi, ngasah kemampuan coding, dan... ya gitu deh!

Teknologi Yang Digunakan

Situs ini menggunakan framework Bootstrap sebagai UI framework. Untuk pengambilan data, dilakukan dengan Javascript menggunakan library jQuery.

Dari mana data-data tersebut diambil?

Data-data tersebut diambil dari data kasus COVID-19 di Indonesia yang disediakan oleh pemerintah, lalu diatur sedemikian rupa agar mudah dibaca.

[RARE CASE] Kenapa Data-nya Tidak Sesuai / Salah?

Data yang ditampilkan sesuai dengan data API pemerintah. Karenanya, data-data yang tampil menjadi tanggung jawab pemerintah. Situs ini tidak mengubah/memanipulasi data apapun, melainkan hanya menampilkan data yang pure berasal dari pemerintah.
